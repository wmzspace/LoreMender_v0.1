# 五关剧情大纲

> 目标：把《典故修补者 · 青囊残卷》整理成适合比赛演示的五关互动叙事流程。每关都有明确的剧情功能、玩家动作、文化表达点和素材需求。

## 总体设定

玩家是“典故修补者”，进入华佗临终前夜的青史投影，附身小徒弟阿吉。玩家不能改变华佗之死，只能决定《青囊经》与医道精神是否能被后人重新看见。

核心主题：

- 历史不可逆，但遗憾可以被重新理解。
- 医书可以散佚，医道可以流入人间。
- 文化表达不是把历史改成爽文，而是在限制中找到价值传承的可能。

## 第一关：牢狱初醒

剧情功能：建立身份、危机和修补规则。

玩家动作：

- 醒在许昌大牢附近，意识到自己附身阿吉。
- 与华佗对话，知道《青囊经》即将失传。
- 第一次选择对师父的态度：忠诚、谦怯或谨慎。

文化表达点：

- 华佗之死不可改变。
- 玩家目标不是“救人”，而是“留道”。

素材需求：

- 插图：夜色牢狱、烛火、竹简、老医者剪影。
- 视频：镜头推进牢房，竹简泛起金墨。
- BGM：低沉古琴、弱弦乐、烛火声。

## 第二关：三人之试

剧情功能：让玩家调查三位托付对象，理解知识传承的风险。

玩家动作：

- 见王济：有权势，可能保护书，也可能封存书。
- 见陈伯：不识字，却真正行医救人。
- 见玄音道人：能避世保存，但可能让知识远离人间。
- 记录第一判断。

文化表达点：

- 知识不是只要“保存”就够了，还要能被人使用。
- 民间流传可能比高阁珍藏更接近文化生命力。

素材需求：

- 插图：街巷、乡野、山门三重空间。
- 视频：三道人影依次出现，金线连向残卷。
- BGM：竹笛、轻打击、探索感。

## 第三关：曹府密谈

剧情功能：把个人选择放进权力压力中，让玩家感到“交书换命”的诱惑。

玩家动作：

- 被曹操召见。
- 面对“交书保师徒平安”的条件。
- 选择沉默、拖延或假意答应。

文化表达点：

- 文化遗产会被权力、私欲和恐惧改变命运。
- 阿吉不是英雄，他只是一个在压力下尽力周旋的小人物。

素材需求：

- 插图：朱漆大堂、青铜灯、桌上竹简。
- 视频：大堂阴影压向小徒弟。
- BGM：低鼓、暗弦、政治压迫感。

## 第四关：青囊抉择

剧情功能：唯一绑定决策，决定结局方向。

玩家动作：

- 回到华佗身边。
- 进入“青囊何归”决策页。
- 选择交给王济、陈伯、玄音道人，或毁去残卷。

文化表达点：

- 选择没有绝对安全，每条路都有代价。
- “最正确”的答案不是最体面，而是最能让医道真正进入人间。

素材需求：

- 插图：残卷、蜡烛、四种命运倒影。
- 视频：卷轴上浮现四种未来。
- BGM：稀疏古琴、心跳式打击、最终抉择感。

## 第五关：千年回响

剧情功能：把选择升华为文化价值表达，并进入结局。

玩家动作：

- 听华佗解释“修补”的真正意义。
- 在“医者本心 / 传承险路 / 选择代价”中做终章理解。
- 进入对应结局和图鉴解锁。

文化表达点：

- 典故修补不是改写事实，而是让事实背后的价值重新发声。
- 玩家完成的不是历史胜利，而是一次文化回响。

素材需求：

- 插图：金墨化光，医道流入村镇与后世。
- 视频：竹简化为千年药灯。
- BGM：古琴、箫、弦乐、铃声，情绪收束。
